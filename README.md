# Matheus Nascimento

Landing page mínima em Astro, pronta para GitHub Pages.

## Rodando localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Editando

O conteúdo central do site está em `src/data/site.ts`.

## Deploy

O deploy é feito pelo workflow em `.github/workflows/deploy.yml`.
